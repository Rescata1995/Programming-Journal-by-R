Introducción a listas!

Los tipos de listas son: ordenadas (ordered list), desordenadas (unordered list) & definidas (definition list)

Desordenadas (unordered list)

Ejemplo con un "Menu de navegación".
  • Elemento 1 a visitar
  • Elemento 2 a visitar
  • Elemento 3 a visitar

Se utilizan cuando el orden de los elementos no influye o no es importante; por ejemplo, una lista de compras; o bien, el mismo menu de navegación.

Ordenadas (ordered list)

Ejemplo con "Orden de aprendisaje de desarrollo web".
  1. HTML
  2. CSS
  3. JavaScript

Se utilizan cuando el orden de los elementos sí es importante (no puede saltarse y debe empezar desde el comienzo establecido);
por ejemplo, un listado de top 10; o bien, el mismo orden de aprendisaje de desarrollo web.

Definidas (definition list)

Continuemos bajo la línea de "Aprendisaje de desarrollo web", como ejemplo.
HTML
HyperText Markup Language: Es un lenguaje de marcado que se usa para estructurar datos en una página web.

CSS
Cascade Style Sheets: Es un lenguaje de diseño que se utiliza para dar estilos al HTML.

JavaScript
Es un lenguaje de programación para dar interactividad a un sitio web.

Se utilizan para hacer una lista de definiciones (tipo diccionario).

Esto es todo sobre los diferentes tipos de listas.
